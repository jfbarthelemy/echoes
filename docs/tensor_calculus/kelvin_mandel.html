1  Kelvin-Mandel notation

Objectives

Before introducing the specific objects of echoes devoted to tensor calculations in isotropic or anisotropic contexts, this tutorial aims at providing the syntax allowing to represent second-order or fourth-order tensors under the form of matrices in the Kelvin-Mandel notation as detailed in Section A.2.

A symmetric \(3Ã—3\) second-order matrix can be transformed in a vector of \(\R^6\) by the function KM consistently with A.14. The inverse is done by invKM.

from echoes import *

Given a \(3Ã—3Ã—3Ã—3\) array c (of type numpy.ndarray) satisfying major and minor symmetries (see Section A.2), the corresponding \(6Ã—6\) matrix C obtained by Kelvin-Mandel transform is calculated by C = KM(c). Conversely, if C is a positive definite matrix, c is calculated by c = invKM(C).
